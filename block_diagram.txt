=============================================================================
                          DETAILED FUNCTIONAL FLOWS
=============================================================================

1. AUTHENTICATION FLOW:
   User -> Login/Register Form -> POST /api/login or /api/register -> Check users.json
   -> Assign User ID -> Return User Object -> Frontend Routes to Dashboard.

2. CONTENT UPLOAD & VERIFICATION FLOW:
   Teacher -> Upload Form -> POST /api/upload/notes (or videos) -> Save to /uploads/
   -> Record in notes.json (status=pending) -> System/Admin verifies via /api/content/verify.

3. REAL-TIME CLASS & TRANSLATION FLOW:
   Teacher -> Starts Class -> POST /api/liveclass/start -> Students GET /api/liveclass -> 
   Student Joins -> Connects via Socket.IO ('join_class_room') -> 
   Student sends Tamil text -> Chat Mod intercepts -> Calls Google Translator -> 
   Translated to English -> Broadcasted to Room via Socket.IO.
