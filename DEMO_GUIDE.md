## 📝 Recording Tips

1. **Resolution**: Record in 1080p (1920x1080)
2. **Duration**: Aim for 5-7 minutes total
3. **Audio**: Consider adding narration explaining each feature
4. **Cursor**: Make sure cursor is visible
5. **Pacing**: Spend 2-3 seconds on each feature to let viewers see it clearly
6. **Transitions**: Click smoothly between pages
7. **Highlight**: Use zoom or highlights for important features
